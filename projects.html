Developing the game

The game engine used to develop the game is "Unity". Done by Dylan Duario.

3D Models Creation

The 3D game assets is made with the software "MagicaVoxel". Done by Eyron Araza.

Character Sprites Creation

The 2D character sprites including animations is made with the software "Pixelorama". Done by Eyron Araza.

Designing the Game UI

The game UI prototype is designed with the software "Figma". Done by Luke Domingo and Omar El Sedek.

Designing Game Levels and Mechanics

The design of game levels including game mechanics is made with the software "Figma". Done by Eyron Araza.
